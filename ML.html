결정 트리의 핵심 개념을 이해하고 실전 문제를 풀어보세요

이 학습 자료는 결정 트리에서 중요한 개념인 엔트로피(Entropy)와 정보 이득(Information Gain)에 대한 이해를 돕기 위해 구성되었습니다. 아래 문제들을 통해 실제 데이터에 엔트로피와 정보 이득을 계산하고 결정 트리의 분기점을 결정하는 과정을 익힐 수 있습니다.

문제 1: 날씨와 피크닉 결정

아래는 14일 동안의 날씨 데이터와 그에 따른 피크닉 진행 여부 기록입니다. 이 데이터를 바탕으로 결정 트리의 첫 번째 분할에 가장 적합한 속성을 찾아보세요.

날짜 전망 온도 습도 바람 피크닉
1일 맑음 높음 높음 약함 아니오
2일 맑음 높음 높음 강함 아니오
3일 구름 높음 높음 약함 예
4일 비 보통 높음 약함 예
5일 비 낮음 보통 약함 예
6일 비 낮음 보통 강함 아니오
7일 구름 낮음 보통 강함 예
8일 맑음 보통 높음 약함 아니오
9일 맑음 낮음 보통 약함 예
10일 비 보통 보통 약함 예
11일 맑음 보통 보통 강함 예
12일 구름 보통 높음 강함 예
13일 구름 높음 보통 약함 예
14일 비 보통 높음 강함 아니오

1. 전체 데이터 집합의 엔트로피를 계산하세요.

2. 각 속성(전망, 온도, 습도, 바람)에 대한 정보 이득을 계산하세요.

3. 계산된 정보 이득을 바탕으로 결정 트리의 첫 번째 분할에 가장 적합한 속성은 무엇인가요?

답안 및 해설

1. 전체 데이터 집합의 엔트로피 계산

전체 14개의 데이터에서 '피크닉=예'는 9개, '피크닉=아니오'는 5개입니다.

P(예) = 9/14 = 0.643, P(아니오) = 5/14 = 0.357

Entropy(S) = -0.643 × log₂(0.643) - 0.357 × log₂(0.357)
Entropy(S) = -0.643 × (-0.637) - 0.357 × (-1.486)
Entropy(S) = 0.409 + 0.531
Entropy(S) = 0.940

따라서, 전체 데이터 집합의 엔트로피는 0.940입니다.

2. 각 속성에 대한 정보 이득 계산

전망(Outlook) 속성에 대한 정보 이득:

전망 = 맑음: 5개 데이터 중 '예'는 2개, '아니오'는 3개
P(예|맑음) = 2/5 = 0.4, P(아니오|맑음) = 3/5 = 0.6
Entropy(맑음) = -0.4 × log₂(0.4) - 0.6 × log₂(0.6) = 0.971

전망 = 구름: 4개 데이터 중 '예'는 4개, '아니오'는 0개
P(예|구름) = 4/4 = 1, P(아니오|구름) = 0/4 = 0
Entropy(구름) = -1 × log₂(1) - 0 × log₂(0) = 0

전망 = 비: 5개 데이터 중 '예'는 3개, '아니오'는 2개
P(예|비) = 3/5 = 0.6, P(아니오|비) = 2/5 = 0.4
Entropy(비) = -0.6 × log₂(0.6) - 0.4 × log₂(0.4) = 0.971

Gain(S, 전망) = Entropy(S) - [(5/14) × Entropy(맑음) + (4/14) × Entropy(구름) + (5/14) × Entropy(비)]
Gain(S, 전망) = 0.940 - [(5/14) × 0.971 + (4/14) × 0 + (5/14) × 0.971]
Gain(S, 전망) = 0.940 - [0.347 + 0 + 0.347]
Gain(S, 전망) = 0.940 - 0.694
Gain(S, 전망) = 0.246

온도(Temperature) 속성에 대한 정보 이득:

온도 = 높음: 4개 데이터 중 '예'는 2개, '아니오'는 2개
Entropy(높음) = -0.5 × log₂(0.5) - 0.5 × log₂(0.5) = 1

온도 = 보통: 6개 데이터 중 '예'는 4개, '아니오'는 2개
Entropy(보통) = -0.667 × log₂(0.667) - 0.333 × log₂(0.333) = 0.918

온도 = 낮음: 4개 데이터 중 '예'는 3개, '아니오'는 1개
Entropy(낮음) = -0.75 × log₂(0.75) - 0.25 × log₂(0.25) = 0.811

Gain(S, 온도) = 0.940 - [(4/14) × 1 + (6/14) × 0.918 + (4/14) × 0.811]
Gain(S, 온도) = 0.940 - [0.286 + 0.394 + 0.232]
Gain(S, 온도) = 0.940 - 0.912
Gain(S, 온도) = 0.028

습도(Humidity) 속성에 대한 정보 이득:

습도 = 높음: 7개 데이터 중 '예'는 3개, '아니오'는 4개
Entropy(높음) = -0.429 × log₂(0.429) - 0.571 × log₂(0.571) = 0.985

습도 = 보통: 7개 데이터 중 '예'는 6개, '아니오'는 1개
Entropy(보통) = -0.857 × log₂(0.857) - 0.143 × log₂(0.143) = 0.592

Gain(S, 습도) = 0.940 - [(7/14) × 0.985 + (7/14) × 0.592]
Gain(S, 습도) = 0.940 - [0.493 + 0.296]
Gain(S, 습도) = 0.940 - 0.789
Gain(S, 습도) = 0.151

바람(Wind) 속성에 대한 정보 이득:

바람 = 약함: 8개 데이터 중 '예'는 6개, '아니오'는 2개
Entropy(약함) = -0.75 × log₂(0.75) - 0.25 × log₂(0.25) = 0.811

바람 = 강함: 6개 데이터 중 '예'는 3개, '아니오'는 3개
Entropy(강함) = -0.5 × log₂(0.5) - 0.5 × log₂(0.5) = 1

Gain(S, 바람) = 0.940 - [(8/14) × 0.811 + (6/14) × 1]
Gain(S, 바람) = 0.940 - [0.463 + 0.429]
Gain(S, 바람) = 0.940 - 0.892
Gain(S, 바람) = 0.048

3. 결정 트리의 첫 번째 분할에 가장 적합한 속성

계산된 정보 이득을 정리하면 다음과 같습니다:

  • 전망(Outlook): 0.246
  • 온도(Temperature): 0.028
  • 습도(Humidity): 0.151
  • 바람(Wind): 0.048

따라서, 전망(Outlook)이 가장 높은 정보 이득(0.246)을 가지므로, 결정 트리의 첫 번째 분할에 가장 적합한 속성입니다.

특히 주목할 점은 '전망 = 구름'일 때의 엔트로피가 0이라는 것입니다. 이는 '구름'이라는 전망 조건에서는 항상 피크닉을 가는 것으로 결정되어 있어, 추가적인 분할이 필요 없음을 의미합니다.

1. 전체 데이터 집합의 엔트로피를 계산하세요.

2. 각 속성(연령대, 성별, 회원 등급, 이전 구매 경험)에 대한 정보 이득을 계산하세요.

3. 가장 높은 정보 이득을 가진 속성은 무엇이며, 해당 속성으로 데이터를 분할한 후 각 하위 집합의 엔트로피는 얼마인가요?

답안 및 해설

1. 전체 데이터 집합의 엔트로피 계산

전체 20개의 데이터에서 '구매=예'는 10개, '구매=아니오'는 10개입니다.

P(예) = 10/20 = 0.5, P(아니오) = 10/20 = 0.5

Entropy(S) = -0.5 × log₂(0.5) - 0.5 × log₂(0.5)
Entropy(S) = -0.5 × (-1) - 0.5 × (-1)
Entropy(S) = 0.5 + 0.5
Entropy(S) = 1

따라서, 전체 데이터 집합의 엔트로피는 1입니다. 이는 데이터가 완전히 균등하게 분포되어 있음을 의미합니다.

2. 각 속성에 대한 정보 이득 계산

연령대 속성에 대한 정보 이득:

연령대 = 20대: 6개 데이터 중 '예'는 3개, '아니오'는 3개
P(예|20대) = 3/6 = 0.5, P(아니오|20대) = 3/6 = 0.5
Entropy(20대) = -0.5 × log₂(0.5) - 0.5 × log₂(0.5) = 1

연령대 = 30대: 7개 데이터 중 '예'는 3개, '아니오'는 4개
P(예|30대) = 3/7 ≈ 0.429, P(아니오|30대) = 4/7 ≈ 0.571
Entropy(30대) = -0.429 × log₂(0.429) - 0.571 × log₂(0.571) ≈ 0.985

연령대 = 40대 이상: 7개 데이터 중 '예'는 4개, '아니오'는 3개
P(예|40대 이상) = 4/7 ≈ 0.571, P(아니오|40대 이상) = 3/7 ≈ 0.429
Entropy(40대 이상) = -0.571 × log₂(0.571) - 0.429 × log₂(0.429) ≈ 0.985

Gain(S, 연령대) = Entropy(S) - [(6/20) × Entropy(20대) + (7/20) × Entropy(30대) + (7/20) × Entropy(40대 이상)]
Gain(S, 연령대) = 1 - [(6/20) × 1 + (7/20) × 0.985 + (7/20) × 0.985]
Gain(S, 연령대) = 1 - [0.3 + 0.345 + 0.345]
Gain(S, 연령대) = 1 - 0.99
Gain(S, 연령대) = 0.01

성별 속성에 대한 정보 이득:

성별 = 여성: 10개 데이터 중 '예'는 6개, '아니오'는 4개
P(예|여성) = 6/10 = 0.6, P(아니오|여성) = 4/10 = 0.4
Entropy(여성) = -0.6 × log₂(0.6) - 0.4 × log₂(0.4) ≈ 0.971

성별 = 남성: 10개 데이터 중 '예'는 4개, '아니오'는 6개
P(예|남성) = 4/10 = 0.4, P(아니오|남성) = 6/10 = 0.6
Entropy(남성) = -0.4 × log₂(0.4) - 0.6 × log₂(0.6) ≈ 0.971

Gain(S, 성별) = Entropy(S) - [(10/20) × Entropy(여성) + (10/20) × Entropy(남성)]
Gain(S, 성별) = 1 - [(10/20) × 0.971 + (10/20) × 0.971]
Gain(S, 성별) = 1 - [0.486 + 0.486]
Gain(S, 성별) = 1 - 0.971
Gain(S, 성별) = 0.029

회원 등급 속성에 대한 정보 이득:

회원 등급 = 골드: 7개 데이터 중 '예'는 7개, '아니오'는 0개
P(예|골드) = 7/7 = 1, P(아니오|골드) = 0/7 = 0
Entropy(골드) = -1 × log₂(1) - 0 × log₂(0) = 0

회원 등급 = 실버: 7개 데이터 중 '예'는 3개, '아니오'는 4개
P(예|실버) = 3/7 ≈ 0.429, P(아니오|실버) = 4/7 ≈ 0.571
Entropy(실버) = -0.429 × log₂(0.429) - 0.571 × log₂(0.571) ≈ 0.985

회원 등급 = 브론즈: 6개 데이터 중 '예'는 0개, '아니오'는 6개
P(예|브론즈) = 0/6 = 0, P(아니오|브론즈) = 6/6 = 1
Entropy(브론즈) = -0 × log₂(0) - 1 × log₂(1) = 0

Gain(S, 회원 등급) = Entropy(S) - [(7/20) × Entropy(골드) + (7/20) × Entropy(실버) + (6/20) × Entropy(브론즈)]
Gain(S, 회원 등급) = 1 - [(7/20) × 0 + (7/20) × 0.985 + (6/20) × 0]
Gain(S, 회원 등급) = 1 - [0 + 0.345 + 0]
Gain(S, 회원 등급) = 1 - 0.345
Gain(S, 회원 등급) = 0.655

이전 구매 경험 속성에 대한 정보 이득:

이전 구매 경험 = 있음: 10개 데이터 중 '예'는 10개, '아니오'는 0개
P(예|있음) = 10/10 = 1, P(아니오|있음) = 0/10 = 0
Entropy(있음) = -1 × log₂(1) - 0 × log₂(0) = 0

이전 구매 경험 = 없음: 10개 데이터 중 '예'는 0개, '아니오'는 10개
P(예|없음) = 0/10 = 0, P(아니오|없음) = 10/10 = 1
Entropy(없음) = -0 × log₂(0) - 1 × log₂(1) = 0

Gain(S, 이전 구매 경험) = Entropy(S) - [(10/20) × Entropy(있음) + (10/20) × Entropy(없음)]
Gain(S, 이전 구매 경험) = 1 - [(10/20) × 0 + (10/20) × 0]
Gain(S, 이전 구매 경험) = 1 - [0 + 0]
Gain(S, 이전 구매 경험) = 1 - 0
Gain(S, 이전 구매 경험) = 1

3. 가장 높은 정보 이득을 가진 속성 및 하위 집합 엔트로피

계산된 정보 이득을 정리하면 다음과 같습니다:

  • 연령대: 0.01
  • 성별: 0.029
  • 회원 등급: 0.655
  • 이전 구매 경험: 1

따라서, 이전 구매 경험이 가장 높은 정보 이득(1)을 가집니다.

'이전 구매 경험'으로 데이터를 분할한 후 각 하위 집합의 엔트로피는:

  • 이전 구매 경험 = 있음: Entropy = 0
  • 이전 구매 경험 = 없음: Entropy = 0

이처럼 두 하위 집합 모두 엔트로피가 0이라는 것은 각 하위 집합이 완전히 '순수(pure)'하다는 것을 의미합니다. 즉, '이전 구매 경험 = 있음'인 고객은 모두 구매를 하고, '이전 구매 경험 = 없음'인 고객은 모두 구매를 하지 않습니다. 이 경우 더 이상의 분할이 필요 없으며, 이전 구매 경험 하나만으로도 구매 여부를 완벽하게 예측할 수 있습니다.

이는 실제 상황에서는 다소 이상적인 케이스이며, 실제 데이터에서는 이렇게 완벽한 분류가 이루어지기 어렵습니다. 하지만 이 예제를 통해 정보 이득이 어떻게 계산되고 어떤 의미를 가지는지를 이해할 수 있습니다.

문제 3: 온라인 게임 승리 예측

인기 온라인 게임 'Battle Royale'에서 플레이어의 특성을 분석하여 게임 승리 여부를 예측하려고 합니다. 다음은 18명의 플레이어 데이터입니다.

플레이어 ID 경험 레벨 아이템 수준 팀 구성 게임 스타일 승리
P1 고급 상급 팀플레이 공격적 예
P2 초급 하급 솔로 방어적 아니오
P3 중급 중급 팀플레이 공격적 예
P4 고급 중급 솔로 공격적 예

  • 기침: 0.413
  • 피로감: 0.289

따라서, 기침이 가장 높은 정보 이득(0.413)을 가지므로, 결정 트리의 첫 번째 분할에 가장 적합한 속성입니다.

기침 속성으로 분할했을 때의 하위 집합:

  • 기침 = 심함: 6개 데이터 (위험도=높음: 5개, 위험도=낮음: 1개)
  • 기침 = 가벼움: 5개 데이터 (위험도=높음: 2개, 위험도=낮음: 3개)
  • 기침 = 없음: 4개 데이터 (위험도=높음: 0개, 위험도=낮음: 4개)

이 결과를 바탕으로 다음과 같은 간단한 결정 규칙을 작성할 수 있습니다:

  1. 만약 기침 = 없음이면, 위험도는 낮음입니다. (100% 확실)
  2. 만약 기침 = 심함이면, 위험도는 높음입니다. (83.3% 확률)
  3. 만약 기침 = 가벼움이면, 추가 검사가 필요합니다:
    • 피로감 속성으로 추가 분할했을 때:
      • 기침 = 가벼움 & 피로감 = 높음인 경우: 3개 데이터 중 위험도=높음은 2개 (66.7%)
      • 기침 = 가벼움 & 피로감 = 낮음인 경우: 2개 데이터 중 위험도=낮음은 2개 (100%)

따라서, 최종 결정 규칙은 다음과 같습니다:

1. 만약 기침 = 없음이면, 위험도 = 낮음
2. 만약 기침 = 심함이면, 위험도 = 높음
3. 만약 기침 = 가벼움이면:
    a. 피로감 = 높음이면, 위험도 = 높음
    b. 피로감 = 낮음이면, 위험도 = 낮음

이 결정 규칙을 통해 새로운 환자의 증상을 바탕으로 질병의 위험도를 예측할 수 있습니다. 예를 들어, 기침이 없는 환자는 위험도가 낮고, 심한 기침이 있는 환자는 위험도가 높을 가능성이 크며, 가벼운 기침이 있는 환자는 피로감의 정도에 따라 위험도가 결정됩니다.

답안 및 해설

1. 전체 데이터 집합의 엔트로피 계산

전체 12개의 데이터에서 '가격 범주=고가'는 4개, '가격 범주=중가'는 4개, '가격 범주=저가'는 4개입니다.
